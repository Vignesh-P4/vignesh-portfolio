# Project Demo Videos

This directory should contain the demo video files for each project. The videos should be:

- **Format**: MP4 (recommended) or GIF
- **Specifications**:
  - Autoplay, muted, and loop enabled
  - Optimized for web (compressed)
  - Aspect ratio: 16:9 (recommended)
  - Duration: 10-30 seconds (looping)
  - File size: Under 5MB per video

## Required Video Files:

1. `fynx-demo.mp4` - FYNX AI Financial Assistant demo
2. `healthify-demo.mp4` - Healthify AI Health Assistant demo
3. `smartagritech-demo.mp4` - SmartAgriTech IoT Agriculture System demo
4. `portfolio-demo.mp4` - Personal Portfolio Website demo

## Video Creation Tips:

- Record screen captures of your applications in action
- Show key features and user interactions
- Keep videos engaging and demonstrate functionality
- Use tools like OBS Studio, ScreenFlow, or Camtasia for recording
- Compress videos using HandBrake or online compressors
- Convert to MP4 with H.264 codec for web optimization

## Fallback:

If video files are not available, the component will gracefully fall back to showing only the static images with the overlay functionality.
